Accessing Sunlab via SSH

Relevant to: CSE 003, CSE 004, CSE 007, CSE 017, CSE 109, CSE 202, CSE 241, CSE 242, CSE 262, CSE 303, CSE 342, CSE 341, CSE 371, CSE 375, CSE 411

What is SSH?

SSH stands for Secure Shell Protocol. It is a network communication protocol that enables two computers to communicate and share data. For this reason, SSH is often used to "login" and perform operations on remote computers. We can use it to remotely connect to Sunlab computers. 

Why do we have to use SSH to access Sunlab?

SunLab is not a website. Therefore, it cannot be connected to via a browser using HTTPS protocol. Instead, connecting to it requires the SSH protocol which provides users secure access to the remote Sunlab machines via an encrypted connection.

How?

Through Command Line or SSH Program like PuTTY

Find MAC CPU Architecture

Relevant to: CSE 003, CSE 004, CSE 007, CSE 109, CSE 202, CSE 216, CSE 271, CSE 342

What is CPU Architecture?

The central "brain" of a computer is the CPU or Central Processing Unit which is commonly referred to as the Processor. Architecture is the designed structure of something. Therefore, CPU Architecture refers to the arrangement of processor components, how they work together, and the distance between transistors. MACs are personal computers built by Apple. 

Why find your MAC's CPU Architecture?

Apple has undertaken a migration from Intel CPUs to an ARM-based processor of their own design under the names M1 and M2. To ensure best performance and battery life, it's important to ensure you are using software that is designed for the chip contained within the Mac being utilized.

How?

Guacamole Remote Desktop Environment

Relevant to: CSE 109, CSE 202, CSE 216, CSE 271, CSE 342, CSE 331, CSE 371, CSE 398, CSE 411

What is Apache Guacamole Environment?

Apache Guacamole is a clientless remote desktop gateway that allows you to connect to computers/servers from anywhere and any time by only using a web browser. It is free and open source and supports protocols like VNC, RDP, and SSH.

Why use Guacamole?

For those that need to remotely access many virtual machines, Guacamole helps lessen the challenge of managing them all in one place. It only requires a browser and doesn’t require any additional plugins.

What is Java?

Java is a high-level, class-based, object-oriented programming language that was created in 1995 by Sun Microsystems but is now owned by Oracle. It is platform independent, meaning that the compiled java code can run on different operating systems. As long as the computer has a Java Runtime Environment (JRE) installed, a Java program can run on it. This concept is encapsulated in their slogan, “Write once, run anywhere”. 

Why use Java?

It is open-source and free, has huge community support and large demand in the job market, works on different platforms, etc

MAGIC GPU Environment

What is the MAGIC GPU Environment?

Graphics processing units (GPUs) are specialized processors originally designed for rendering graphics on a display. They have recently been made generic enough to work with scientific computing problems. 

“Machine-Learning Artificial-Intelligence GPU Instructional Cluster” (MAGIC) is the CSE department’s GPU used for machine-learning and artificial intelligence. It can be accessed via a web browser. It runs JupyterHub which allows for easy interaction and coding.

Due to their massively parallel architecture, GPUs enable the completion of computationally intensive tasks much faster than conventional CPUs.

SSH X-Forwarding

Relevant to: CSE 109, CSE 202, CSE 216, CSE 262, CSE 271, CSE 303, CSE 342

What is SSH X-Forwarding?

SSH X-forwarding allows you to forward programs that use a graphical user interface through your remote SSH connection to your local computer. The X refers to the X Window System—an open source graphics protocol that provides a basic framework for creating custom GUIs that can display graphics on both local and remote display devices. 

Why use SSH X-Forwarding?

SSH only allows text-based interaction between the user and the server programs. X forwarding is a useful and reliable tool as it allows the user to run graphical programs on remote machines securely

Visual Studio Code

Relevant to: CSE 003, CSE 004, CSE 007, CSE 017, CSE 109, CSE 202, CSE 216, CSE 262, CSE 271, CSE 303, CSE 331, CSE 342, CSE 375, CSE 411

What is Visual Studio Code?

Visual Studio Code (VS Code) is a standalone source code editor that runs on Windows, macOS, and Linux. It has extensions to support just about any programming language. Features include support for debugging, syntax highlighting, intelligent code completion, snippets, code refactoring, and embedded Git. 

Why use Visual Studio Code?

Visual Studio Code combines the simplicity of a source code editor with powerful developer tooling, like IntelliSense code completion and debugging. It's free, uses extensions to handle multiple languages, and once you learn how to use it (user snippets) it really accelerates your productivity.
