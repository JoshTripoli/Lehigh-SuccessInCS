Version Control


Git

Relevant to: programming in any language

Git is a version control system that allows developers to keep track of changes to their code and collaborate with others on projects. With Git, developers can create and maintain different versions of their code, switch between different branches of code, and merge changes made by multiple contributors.

Programming Languages


C

Relevant to: CSE 109, CSE 202, CSE 303, CSE 331, ECE 205

Developed by Dennis Ritchie in the early 1970s, C has since become one of the most popular programming languages in the world. C provides low-level access to the system's hardware, which makes it ideal for writing applications that require high performance and low-level control.

C++

Relevant to: CSE 109, CSE 202, CSE 303, CSE 331, ECE 205

C++ is an object-oriented programming language that was created as an extension of the C language. It was developed by Bjarne Stroustrup in the early 1980s and provides features such as classes, templates, and inheritance, which make it possible to write complex and scalable applications.

C#

Relevant to: Unity

C# (pronounced C sharp) is a modern programming language developed by Microsoft in the early 2000s. It is designed to be easy to learn and use, and is often used to develop Windows desktop applications, web applications, and games.

Dart

Relevant to: CSE 216

Dart is a modern, object-oriented programming language developed by Google. It is often used with Flutter for building mobile and web applications.

Excel VBA

Relevant to: Excel and other Office programs

Excel VBA (Visual Basic for Applications) is a programming language used to create and automate tasks in Microsoft Excel. It allows users to write code to manipulate data, create custom functions, and automate repetitive tasks, making it a powerful tool for data analysis and management.

Java

Relevant to: CSE 003, CSE 004, CSE 007, CSE 241, CSE 428, CSE 431


Developed in the mid-1990s, Java is known for its simplicity, platform independence, and robustness. One of the key features of Java is its "write once, run anywhere" philosophy, which allows code to be written once and executed on any platform that supports Java. With its widespread adoption and mature ecosystem, Java remains one of the most popular programming languages today.

Python

Relevant to: ENGR 010, CSE 012, CSE 325/CSE 425, CSE 326/CSE 426 , CSE 347, CSE 349/449, CSE 342, CSE 398


Python is a popular high-level programming language that emphasizes simplicity, readability, and ease of use. Developed in the late 1980s, Python has grown to become one of the most widely used programming languages today, thanks to its versatility and ease of learning. Python is used in a wide variety of applications, including web development, data analysis, artificial intelligence, and scientific computing.

R

Relevant to: CSE 160

R is an open-source programming language used for statistical computing and data analysis. It provides a wide range of tools for data manipulation, visualization, and modeling, making it a popular choice among data scientists, researchers, and statisticians. 

Rust

Relevant to: CSE 262

Rust is a systems programming language that emphasizes performance, safety, and concurrency. It has gained popularity in recent years for its ability to prevent common programming errors and improve code reliability.

Scheme

Relevant to: CSE 262

Scheme is a classic programming language in the Lisp family. It emphasizes functional programming and domain-specific languages but adapts to other styles. Known for its clean and minimalist design, Scheme is one of the longest-lived and best-studied dynamic languages, and has many fast and portable implementations.

SQL

Relevant to: CSE 216, CSE 241, CSE 341

SQL (Structured Query Language) is a programming language used to manage and manipulate relational databases. It allows users to insert, update, delete, and query data in a database, making it an essential tool for managing large amounts of information in various industries. 

Web Development

HTML

Relevant to: CSE 216, CSE 264, CSE 331


A markup language that serves as the foundation of most websites on the internet. It provides a set of predefined tags that developers use to define the structure and content of their web pages.

CSS

Relevant to: CSE 216, CSE 264, CSE 331


A stylesheet language that allows developers to style the elements defined in HTML and define how they are displayed on the webpage.

Javascript

Relevant to: CSE 216, CSE 264, CSE 331


A programming language used to create dynamic and interactive effects in web pages and web applications. 

React

Relevant to: CSE 216

React is a JavaScript library for building user interfaces. It was developed by Meta and is widely used by web developers for creating reusable UI components and managing the state of web applications. 


Game Development


Unity

Unity is a popular cross-platform game engine used for developing games and interactive 3D applications. It offers a powerful set of tools for creating immersive and engaging experiences across a variety of platforms, including PC, mobile, and consoles. With a vast community of developers and a wealth of online resources, Unity is a top choice for game development.
